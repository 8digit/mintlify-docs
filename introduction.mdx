title: "What is Knouds? Visual AI pipeline editor overview"

---

Knouds is a visual drag-and-drop canvas where you build AI pipelines by connecting nodes. Each node represents a step in your workflow — accepting input, transforming data with an AI model, or returning output. When your pipeline is ready, you deploy it as a REST API endpoint and call it from any application.

## What you can build

Knouds supports generation across multiple media types and AI providers:

- **Images** — text-to-image and image-to-image generation
- **Video** — text-to-video and image-to-video generation
- **Music** — AI audio and music generation
- **LLM** — chat, vision, and reasoning workflows using language models
- **Custom pipelines** — chain any combination of the above, add HTTP request nodes, file uploads, and list processing

Every workflow you build can be deployed as a REST API, letting you integrate AI generation into your own products, scripts, and automations.

## Who Knouds is for

Knouds is designed for developers, product teams, and creators who want to prototype and ship AI-powered features without writing integration code from scratch. If you've ever spent hours wiring together API calls to multiple AI providers, Knouds replaces that with a visual canvas and a single endpoint.
